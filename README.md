# 💸 FinTrack - Finance Dashboard

A modern and responsive personal finance dashboard built using **React, Zustand, and Tailwind CSS**.

---

## 🚀 Features

### 📊 Core Features
- Add and manage transactions (income & expenses)
- Real-time balance, income, and expense calculation
- Interactive charts (line + pie)
- Category-wise expense tracking

### 👥 Role-Based Access
- **Viewer** → Can only view data
- **Admin** → Can add and edit transactions

### 📈 Insights
- Highest spending category
- Monthly comparison of transactions
- Savings analysis (income vs expenses)

---

## ⚡ Optional Enhancements (Implemented)

- 🌙 Dark Mode (with persistence)
- 💾 Data persistence using Local Storage
- 📤 Export transactions:
  - CSV
  - JSON
- 🔍 Advanced filtering:
  - By type (income/expense)
  - By category
  - Search functionality
- 📱 Fully responsive UI (mobile + desktop)
- 🎯 Mock API simulation (optional)

---

## 🛠️ Tech Stack

- React (Vite)
- Zustand (state management)
- Tailwind CSS (UI styling)
- Lucide Icons
